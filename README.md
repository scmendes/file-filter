# file-filter

Aplicação que permite leitura e operacoes de consulta em arquivo CSV

### Overview da arquitetura
- Java 8 e JUnit 4

### Classe FileFilter.java: 
A classe br.com.involves.java.FileFilter é responsável pela inicialização da aplicação.


### Classe FileFilterTest.java
A classe br.com.involves.test.FileFilterTest deve ser invocada para execução dos testes unitários.
Estão contemplados 3 testes, sendo cada um deles direcionado a um dos comandos (count all, count distinct, filter).
